### 오늘 인상깊었던 파이썬 코드

```python
list[start : end : step]
# start : 슬라이싱 위치
# end : 슬라이싱 끝나는 위치(미포함)
# step : 보폭(음수 시 거꾸로, 절대값 : 건너 뛰는 값)
```



```python
word = input()
result = [-1] * len(string.ascii_lowercase)
for i in range(len(word)):
    val = ord(word[i]) - ord('a')  # 문자를 숫자로 변환하는 함수
    if result[val] == -1:
        result[val] = i
print(' '.join([str(word) for word in result])) # join 잘 기억하기
```
